eMonitor - Administrationsbereich

Hier können Sie alle Einstellungen der Software konfigurieren und überwachen. Die Software gliedert sich in unterschiedliche Bereiche:

Webinterface

Das Webinterface liefert in unterschiedlichen Bereichen die Möglichkeit der kompletten Einsatzabwicklung. Im Frontend kann über eine Kartenansicht die Einsatzstelle vorab erkundet werden. Es können für den Einsatz relevante Daten auf der Karte eingesehen und in Telegrammform ausgedruckt werden.

Faxüberwachung

Die Faxüberwachung kontrolliert in konfigurierbaren Zeitintervallen ein definierbares Verzeichnis auf neue Faxnachrichten, die in verschiedenen Dateiformaten eingeben können. Unterstützt werden grundsätzlich PDF, TIFF und JPEG (PDF und TIFF auch mehrseitig).

Monitorsteuerung

Die Monitorsteuerung kann über einen eigenen Klient automatisch die Monitore steuern. Dabei reicht es aus, die Klient-Software zu starten und eine ID zu vergeben. Ab sofort wird die Monitorsteuerung bei Events den Monitor mit beschicken und automatisch die korrekte Anzeige liefern.

Eingesetzte Software

  • python
  • Tornado - Webserver
  • Flask - Microframework
  • jQuery
  • jQuery - UI
  • jQuery - Layout
  • jQuery - Accordion
  • Leaflet - Kartendarstellung
  • OpenStreetMap - Karteninformationen

Sämtliche Software liegt OpenSource in verschiedenen Lizenzmodellen vor, kann aber immer kostenfrei genutzt werden. Es wurde darauf geachtet, dass die Betriebssystemunabhängigkeit gegeben ist. Wenn Python vom System unterstützt wird, kann eMonitor bertieben werden.
