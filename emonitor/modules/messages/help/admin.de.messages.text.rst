Text-Mitteilungen
=================

Mitteilungen vom Typ *Text* sind der einfachste Mitteilungstyp. Der angegebene Text wird direkt auf dem Monitor
dargestellt. Zur besseren Formatierung wird der eingegebene Text nach dem *restructuredText* Format formatiert, so dass
die grundlegenden HTML-Elemente automatisch erzeugt werden.

Dieser Mitteilungstyp kann auch als Default-Mitteilung verwendet werden.
