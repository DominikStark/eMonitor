############
Mitteilungen
############

Mit eMonitor können Mitteilungen erstellt werden, die immer dann angzeigt werden, wenn keine aktiven Einstäze vorliegen.
Es stehen verschiedene Mitteilungstypen zur Verfügung, die auch als Standard angezeigt werden können, wenn sonst keine
Information für die Monitore zur Anzeige vorliegt.
