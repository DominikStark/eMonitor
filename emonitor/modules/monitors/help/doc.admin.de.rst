####################
Monitorkonfiguration
####################

In der Monitorkonfiguration können das Layout und die unterschiedlichen Monitore definiert werden, die die Inhalte von
eMonitor darstellen. Monitore sind dabei unabhängig von einander und können für die verschiedenen Events
unterschiedliche Inhalte darstellen - angepasst auf den Aufstellungsort.

Für die Monitore stehen *Themes* zur Verfügung, die die Anzeige in Form und Farbe zusätzlich beeinflussen können.
Zusätzlich gibt es einen Bereich, in dem die aktuell geplanten Layout-Änderungen der Monitore angezeigt werden.
