############
Straßenkarte
############

Es können verschiedene Karten definiert werden, zwischen denen im Frontend umgeschaltet werden kann. Dabei sind sowohl
Offline- als auch Online-Karten möglich.

Bei Offline-Karten werden die *Tiles* automatisch bei bestehender Internetverbindung aus dem Internet geladen.
Zusätzlich kann definiert werden, welche der Karten als Standardkarte genutzt werden soll.
