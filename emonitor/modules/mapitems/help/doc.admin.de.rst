#############
Kartenobjekte
#############

Auf der Karte können unterschiedliche Objekte dargestellt werden. Standardmäßig können diese Objekte direkt von
OpenStreetMap importiert werden. Dafür sind verschiedene Filter in eMonitor integriert.

Zusätzlich können auch direkt auf der Karte Objekte dargestellt werden. Die Definition erfolgt dabei über dieses
Module. Alle Kartenobjekte müssen eine genaue Koordinate besitzen, dass sie an der korrekten Stelle dargestellt werden
können.
